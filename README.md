# Hammerspoon Control Scroll Arrows

Converts Control + mouse or trackpad scrolling into Control + Arrow keystrokes.

## Install

```sh
brew install --cask hammerspoon
git clone https://github.com/varundeepsaini/hs-scroll-arrows.git
mkdir -p ~/.hammerspoon
cp hs-scroll-arrows/control-scroll-arrows.lua ~/.hammerspoon/init.lua
open -a Hammerspoon
```

Grant Hammerspoon Accessibility and Automation permissions when prompted, then reload its configuration.
